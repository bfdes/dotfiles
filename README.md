# dotfiles

My dotfiles and installation script for software development on Mac.

## Requirements

Setup should be compatible with any machines running macOS Catalina.

## Usage

Within the repository directory:
* Make the script executable with `chmod +x ./install.sh` 
* Run the script with `install.sh`

Some commands require root permissions, so you may be asked for your password.

Note that, as far as possible, non-idempotent operations are commented out by default.
